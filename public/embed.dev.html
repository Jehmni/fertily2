Fertility Chat Widget Integration (Development)

Development Configuration

Configure the widget endpoint:

To add the widget to your site during development, add this script:

<script>
window.FERTILITY_WIDGET_CONFIG = {
    apiUrl: "your-api-url-here"
};
</script>
<script src="/widget.dev.js"></script>

Live Demo

The chat widget should appear as a floating button in the bottom-right corner of this page.
